Settings

            
Tune what's flagged "stale", which MEL keys are critical, and the per-type splits used in the Critical slide. Changes take effect immediately.

            

          


          

            
Remove a report snapshot

            

              Deletes the one report date selected in the Snapshot tab’s date control (⤒ Latest / calendar dropdown) — the same as the red Delete snapshot control there. It removes the workbook and analysis for that day from storage, updates history, and rebuilds work-order and MEL data. This cannot be undone.
            

            

              
            

          


          

            
Email ingest security

            

              The Vehicle ETIC workbook is ingested only when an email reaches your Cloudflare Email Routing address with the attachment.
              If ALLOWED_SENDERS is * in Worker vars (this repo’s default), any sender can trigger ingest — spam, misconfigured auto-forward, or another mailbox on the same rule.
              For production, set ALLOWED_SENDERS to a comma-separated allowlist (e.g. deploy with explicit vars).
              In observability logs, search for etic_workbook_email_received to see from, to, and dateKeySource when a file lands.
            

          


          

            
Work-order remark staleness

            
A WO is considered "stale" when its last remark is older than this many days. Tunable per MEL status so below-MEL keys can be held to a tighter cadence.
